# Flowchart Threaderer

**Програмний інструмент для візуального проектування та тестування багатопоточних програм за допомогою блок-схем**

## Опис проекту

Flowchart Threaderer — це десктопний додаток, який дозволяє користувачу створювати, редагувати та тестувати поведінку багатопоточних програм шляхом малювання блок-схем. Кожна блок-схема відповідає окремому потоку. Інструмент автоматично транслює набір блок-схем у виконуваний код на C# та проводить автоматичне тестування, враховуючи можливу недетермінованість багатопоточних програм.

Проект повністю відповідає технічному завданню лабораторної роботи.

## Виконані вимоги

### Пункт 1. Візуальний редактор блок-схем
- Графічний інтерфейс для створення та редагування блок-схем.
- Підтримка від 1 до 100 потоків (вкладки).
- Кожна блок-схема може містити до 100 блоків.
- Підтримка циклів в орієнтованих графах.
- Доступні дії в блоках:
  - `V1 = V2` (присвоєння змінних)
  - `V = C` (присвоєння константи)
  - `INPUT V`
  - `PRINT V`
- Умови розгалуження:
  - `V == C`
  - `V < C`
  - `V > C`
- Можливість збереження/завантаження проєктів.

### Пункт 2. Автоматична генерація коду
- Трансляція набору блок-схем у повноцінну багатопоточну програму на **C#**.
- Використання `System.Threading.Thread` для кожного потоку.
- Спільна пам’ять (`int[] V = new int[100]`).
- Генерація читабельного коду з можливістю перегляду перед виконанням.

### Пункт 3. Автоматичне тестування (реалізовано)
- Створення та управління набором тестів (вхідні дані + очікуваний вивід).
- Автоматична компіляція та виконання згенерованого коду.
- Підтримка **недетермінованих програм** — перебіг різних варіантів виконання.
- Можливість задати глибину тестування (`K ≤ 20`).
- Виведення статистики: відсоток перевірених виконань, що відповідають тесту.
- Окреме вікно тестування з можливістю переривання процесу.

## Технологічний стек

- **Мова:** C# (.NET Framework / WPF)
- **UI:** Windows Presentation Foundation (WPF)
- **Графіка:** Кастомні контролі (`BlockControl`, `ConnectionControl`)
- **Генерація коду:** Динамічна трансляція блок-схем у C#
- **Компіляція та виконання:** `Microsoft.CodeAnalysis.CSharp` + Reflection / Process
- **Архітектура:** MVVM-подібна, розділення на `Models`, `Controls`, `Services`

## Структура проєкту
FlowchartThreaderer/
├── Controls/           # Кастомні елементи керування (блоки, з'єднання, канва)
├── Models/             # Моделі даних (TestCase, Block тощо)
├── Services/           # Логіка: генерація коду, тестування
├── MainWindow.xaml     # Головне вікно з вкладками потоків
├── TestWindow.xaml     # Вікно автоматичного тестування
├── CodePreviewWindow.xaml # Перегляд згенерованого коду
└── ...

## Можливості програми

- Додавання/видалення потоків
- Малювання блок-схем (drag & drop, з'єднання)
- Збереження проєкту у файл
- ЗАвантаження проекту з файлу
- Генерація та перегляд C# коду
- Створення тестових наборів
- Автоматичне тестування з підтримкою недетермінізму
- Інтуїтивний графічний інтерфейс

## Як запустити

1. Відкрийте рішення `FlowchartThreaderer.sln` у Visual Studio.
2. Встановіть NuGet-пакет (якщо потрібно): `Microsoft.CodeAnalysis.CSharp`.
3. Запустіть проект (`F5`).
